Na aula de hoje vamos aprender integrar as ferramentas
de Git e GitHub para poder trabalhar em projetos de 
forma mais fácil e dinâmica.

Hoje será a primeira aula onde conectaremos o
repositório online (que está no GitHub) com o 
repositório da minha máquina.

Edição realizada pelo GitHub

Comandos aprendidos na aula de hoje:
git push -> serve para enviar os arquivos da máquina local para o repositório online no GitHub
git pull -> serve para atualizar o repositório local com os dados do repositório online no GitHub
